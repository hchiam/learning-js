All text on this page will automatically translate with the selection in the selection box generated from Google's "translation API". Unless you add class="notranslate" on the body, and class="translate" on the target.

This element with id="myDivIDMakeIntoSelectBox" will turn into a selection box.

// This code is needed to generate the selection box:
function googleTranslateElementInit() { // this function name will be recognized by the API
    google.translate.TranslateElement({pageLanguage: 'en'}, 'myDivIDMakeIntoSelectBox');
}

<script src="https://translate.google.com/translate_a/element.js?cb=googleTranslateElementInit"></script>
